Color Memory Game

The Goal of the game is to remember the order of the button presses then repeat the sequence by clicking the buttons in the same order.

The pace will increase after every correct sequence. There are a total of 9 sequences and you have 3 lives. Losing a life does not reset the current sequence. You also have 9 seconds to complete the current sequence you're on.

Be careful though, because if you run out of time, its Game Over.

Good Luck!
